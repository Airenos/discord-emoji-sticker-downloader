# Discord Emoji & Sticker Downloader 📥

用于批量导出当前 Discord 账号可访问服务器中的自定义 Emojis 和 Stickers，并打包为 ZIP 下载到本地。

## 功能

- 批量导出服务器自定义 Emojis
- 批量导出服务器 Stickers
- 支持选择部分项目或全选
- 自动打包为 ZIP
- 右下角浮动按钮支持拖动，避免遮挡 Discord 页面

## 重要说明

本脚本会在本地使用当前 Discord Web 登录会话的认证信息，并仅用于向 Discord API 请求你当前账号可访问服务器的 Emoji / Sticker 元数据。

本脚本不会把认证信息上传到作者服务器，也不包含统计、广告或远程控制代码。

但由于该行为涉及 Discord Web 会话和非官方使用方式，请在安装前阅读源码，并自行承担账号与平台规则风险。

## 使用范围

本工具仅适用于你已加入、且当前账号可访问的 Discord 服务器。

请勿将导出的资源用于未经授权的商业用途。
